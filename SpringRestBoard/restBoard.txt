* HTTP 메서드 동작
	
	GET		: 조회(select)
	POST	: 추가(insert / create)
	PUT		: 전체수정(update)
	PATCH	: 부분수정(update)
	DELETE	: 삭제(delete)
	
	
* REST방식으로 URI 설계

	/ 작업명 / 기본키 + 메서드 + 데이터
	
	작업명 : 요청하려는 작업의 종류(이름)
	기본키 : 요청작업에 따른 대상의 기본키값
	메서드 : HTTP메서드 종류
	데이터 : 작업을 수행할 때 필요한 JSON 데이터
	
	* 게시판 생성
	
	글쓰기 : /boards + 데이터 (POST)
	글조회 : /boards/1000 (GET)
	글수정 : /boards/ (UPDATE)
	글삭제 : /boards/ (DELETE)
